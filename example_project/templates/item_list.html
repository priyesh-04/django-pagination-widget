💡 Usage Example

In your Django templates, simply use:

{% verbatim %} {% load pagination_tags %} {% pagination_widget page_obj %} {% endverbatim %}

Or with a custom page range:

{% verbatim %} {% pagination_widget page_obj custom_page_range %} {% endverbatim %}

🔧 Features Demonstrated

  • Smart Ellipsis: Notice how page numbers are hidden with "…" when there are many pages
  • Active State: Current page is highlighted
  • Navigation: Previous/Next buttons with inline SVG icons
  • Responsive: Try resizing your browser window
  • Accessibility: Proper ARIA labels and semantic HTML
